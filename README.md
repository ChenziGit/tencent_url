# QQ音乐无损解析 API (QQ Music API)

基于 Python 和 Flask 构建的 QQ 音乐无损音质解析与搜索后端接口。
项目支持获取多种高质量音频链接（包括 FLAC、Master、全景声 Atmos、320k 等）、歌词解析，以及关键词搜索歌曲和歌单功能。

---

## ✨ 核心特性

- **多音质音频支持**: 从基础 `m4a/128k` 到 高品质 `320k`，再到无损 `ape/FLAC` 甚至 `Master`/`Atmos` 等更高规格音质。
- **全方位歌曲信息**: 不仅提供音频播放链接，还返回歌曲名、专辑信息、歌手大图等。
- **双语歌词解析**: 内置支持原文歌词和翻译歌词获取。
- **无依赖搜索 API**: 提供关键词搜索歌曲和歌单的专属接口，搜索功能**无需**配置 Cookie 即可使用。

## 🚀 快速运行部署

### 1. 环境要求
- **Python** >= 3.x

### 2. 安装依赖
克隆或下载源码后，在项目根目录运行以下命令安装所需依赖库：
```bash
pip install -r requirements.txt
```

### 3. 配置 Cookie (必需)
由于腾讯的限制，若要解析任意歌曲、特别是 **VIP 专属歌曲**以及**无损级别**等高音质音频，**必须获取并配置对应的 QQ 音乐（通常需要会员）Cookie**。

1. 在浏览器网页端登录 [QQ 音乐 (y.qq.com)](https://y.qq.com/) 
2. 使用开发者工具抓包获取请求携带的 `Cookie` 字符串。
3. 打开本项目中的 `app.py` 文件，找到全局变量 `cookie_str`（约第 11 行）。
4. 将获取到的 Cookie 填入：
   ```python
   cookie_str = '在此处填入你抓取到的QQ音乐Cookie'
   ```

### 4. 启动服务
```bash
python app.py
```
运行成功后，默认服务将监听在该地址：`http://0.0.0.0:5122`

## 📖 接口文档 API 参考

### 1. 解析单曲接口

传入 QQ 音乐客户端或者是网页端获取到的单曲分享/复制链接，通过该 API 完成各音质和信息的全面解析。

- **请求路径**: `/song`
- **请求方式**: `GET`
- **请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :---: | :--- |
| `url` | `string` | 是 | 需解析的 QQ 音乐分享链接地址（支持短链及主站带 id 的链接） |

- **返回数据结构与说明**:
  - `song`: 包含歌曲核心信息（歌名 `name`、专辑 `album`、歌手 `singer`、歌曲封面图片大图 `pic`、歌曲ID等）。
  - `lyric`: 包含 Base64 解码后的原文歌词 `lyric` 与翻译歌词 `tylyric`。
  - `music_urls`: 返回多种音质类型的播放与下载链接（示例包含 `128`, `320`, `flac`, `master`, `atmos_2`, `aac_192`, `ogg_320` 等），如遇 VIP 或权限不足时对应音质 URL 处理为 `None`。

---

### 2. 搜索歌曲接口

按搜索关键词返回歌曲的匹配列表。支持分页展示数据。

- **请求路径**: `/search`
- **请求方式**: `GET`
- **请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :---: | :--- |
| `keyword` | `string` | 是 | 搜索的请求关键词（支持歌曲名称、歌手名称等） |
| `page` | `int` | 否 | 页码控制（默认值：`1`） |
| `limit` | `int` | 否 | 每页返回的项数（默认值：`20`） |

- **返回示例**:
```json
{
  "code": 200,
  "msg": "success",
  "data": [
    {
      "songmid": "0039MnYb0qxYhV",
      "songname": "晴天",
      "singer": "周杰伦",
      "albumname": "叶惠美",
      "url": "https://y.qq.com/n/ryqq/songDetail/0039MnYb0qxYhV" 
    }
  ]
}
```

---

### 3. 搜索歌单接口

按搜索关键词返回匹配的精选歌单列表结果。支持分页。

- **请求路径**: `/search/playlist`
- **请求方式**: `GET`
- **请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :---: | :--- |
| `keyword` | `string` | 是 | 搜索相关歌单的关键词 |
| `page` | `int` | 否 | 页码控制（默认值：`1`） |
| `limit` | `int` | 否 | 每页返回的项数（默认值：`20`） |

- **返回示例**:
```json
{
  "code": 200,
  "msg": "success",
  "data": [
    {
      "dissid": "7039749142",
      "dissname": "百听不厌的周杰伦",
      "creator": "今晚月色很美",
      "imgurl": "http://qpic.y.qq.com/music_cover/.../300?n=1",
      "song_count": 99,
      "listennum": "3.9亿"
    }
  ]
}
```
